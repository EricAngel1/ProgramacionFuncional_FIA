## Cómo ejecutar el servidor

1. Abrir una **terminal o consola**.

2. Iniciar el intérprete de Prolog:

```
swipl
```

3. Cargar el archivo del servidor:

```
consult('agentesoftWeb.pl').
```

4. Iniciar el servidor con el siguiente comando:

```
servidor(8000).
```

---

## Acceder

Una vez iniciado el servidor, abrir un navegador web y acceder a:

```
http://localhost:8000
```
